# More-C-MORE

## Run

```
python3 run.py --input_path <input_path> --output_path <output_path>
```

### Input file format

```
[
    {
    'question': '',
    'positive_ctxs': [{'title': '', 'text': ''}],
    'negative_ctxs': [],
    'hard_negative_ctxs': [{'title': '', 'text': ''}],
    'answers': [''],
    'question_word': '',
    'answer_start': [int]
    }
]
```
